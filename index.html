Types of licenses

  1. Proprietary
  2. Public domain
  3. Protective or Copyleft
  4. Permissive

Proprietary licenses

  • For commercial software
  • Not negotiable
  • Source code is not available
  • Redistribition is not allowed

Public domain

Open source software licensing

Copyleft vs Permissive


The main difference is
under what conditions you can distribute derivative software.

Copyleft licenses

GNU General Public License 
(GNU GPL or GPL)


Created in 1989 by Richard Stallman

  • Making source code available is a MUST
  • Any derivative software MUST be licensed under GPL

Permissive licenses

  • MIT
  • Apache
  • BSD

MIT

  • Short, clear, easy
  • Anyone can rebrand and sell your software

Apache

  • Users can’t remove existing copyright, patent, trademarks
  • Best patent protection among the permissive licenses
  • You must notify about any changes in the software
